同一时间，发生了人类变化成盐柱的诡异现象（一般称之为“盐害”），光是曰本地区的死亡人数便估计多达八千万人。文明社会在一瞬间崩溃，劫后余生的人们逃到农村，过着自给自足的贫乏生活……
